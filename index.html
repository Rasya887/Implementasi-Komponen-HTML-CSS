Implementasi Komponen HTML & CSS

1. Struktur Dasar HTML

HTML memiliki struktur dasar yang terdiri dari tag <!DOCTYPE>, <html>, <head>, dan <body>.

<!DOCTYPE html>
<html lang="id">
  <head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Judul Halaman</title>
  </head>
  <body>
    Konten halaman di sini
  </body>
</html>

2. Heading dan Paragraf

HTML menyediakan tag <h1> hingga <h6> untuk judul dan <p> untuk paragraf.

<h1>Judul Utama</h1>
<h2>Sub Judul</h2>
<p>Ini adalah sebuah paragraf dengan teks yang menjelaskan sesuatu.</p>

3. Tautan dan Gambar

Tautan menggunakan tag <a> dan gambar menggunakan <img>.

<a href="https://example.com" target="_blank">Kunjungi Website</a>
<img src="gambar.jpg" alt="Deskripsi Gambar" width="300" height="200">

4. List (Daftar)

Ada dua jenis daftar: tidak berurutan (<ul>) dan berurutan (<ol>), yang berisi item <li>.

<ul>
  <li>Apel</li>
  <li>Jeruk</li>
  <li>Mangga</li>
</ul>

<ol>
  <li>Persiapan bahan</li>
  <li>Proses memasak</li>
  <li>Penyajian</li>
</ol>

5. Tabel

Tabel dibuat menggunakan tag <table>, baris <tr>, kolom <td>, dan judul kolom <th>.

<table>

      <th>Kota</th>
    </tr>
  </thead>

    </tr>
    <tr>
      <td>Sari</td>

    </tr>
  </tbody>

7. CSS Selector & Properties

CSS menggunakan selector untuk memilih elemen HTML dan menerapkan styling. Berikut adalah jenis-jenis selector utama:

A. Selector Dasar

/* Element Selector */
h1 { color: blue; }

/* Class Selector */
.highlight { background-color: yellow; }

/* ID Selector */
#header { font-size: 24px; }

/* Universal Selector */
* { margin: 0; padding: 0; }

B. Selector Kombinasi

/* Descendant Selector */
div p { color: gray; }

/* Child Selector */
ul > li { list-style: none; }

/* Adjacent Sibling */
h1 + p { margin-top: 0; }

/* General Sibling */
h1 ~ p { font-style: italic; }

C. Pseudo-classes & Pseudo-elements

/* Pseudo-classes */
a:hover { color: red; }
input:focus { border-color: blue; }
li:nth-child(odd) { background: #f0f0f0; }

/* Pseudo-elements */
p::first-line { font-weight: bold; }
p::before { content: "→ "; }

8. CSS Box Model

Box Model adalah konsep fundamental dalam CSS yang menjelaskan bagaimana elemen HTML ditampilkan sebagai kotak.

.box {
  /* Content area */
  width: 200px;
  height: 100px;
  
  /* Padding - ruang dalam */
  padding: 20px;
  
  /* Border - garis tepi */
  border: 2px solid #333;
  
  /* Margin - ruang luar */
  margin: 10px;
  
  /* Box-sizing untuk mengontrol perhitungan ukuran */
  box-sizing: border-box; /* width termasuk padding & border */
}

9. CSS Layout - Flexbox

Flexbox adalah sistem layout yang memudahkan pengaturan elemen dalam satu dimensi (horizontal atau vertikal).

.flex-container {
  display: flex;
  justify-content: space-between; /* main axis alignment */
  align-items: center;           /* cross axis alignment */
  flex-direction: row;           /* atau column */
  flex-wrap: wrap;               /* untuk wrapping */
  gap: 1rem;                     /* jarak antar item */
}

.flex-item {
  flex: 1;        /* grow, shrink, basis */
  flex-grow: 1;   /* kemampuan membesar */
  flex-shrink: 0; /* kemampuan mengecil */
  flex-basis: 200px; /* ukuran dasar */
}

10. CSS Grid Layout

CSS Grid adalah sistem layout dua dimensi yang sangat powerful untuk membuat layout kompleks.

.grid-container {
  display: grid;
  grid-template-columns: repeat(3, 1fr); /* 3 kolom sama besar */
  grid-template-rows: auto;
  gap: 1rem;                             /* jarak antar grid */
  
  /* Atau dengan area template */
  grid-template-areas: 
    "header header header"
    "sidebar main main"
    "footer footer footer";
}

.grid-item {
  grid-column: 1 / 3;  /* dari kolom 1 sampai 3 */
  grid-row: 1 / 2;     /* dari row 1 sampai 2 */
}

11. CSS Colors & Gradients

CSS menyediakan berbagai cara untuk mengatur warna dan gradasi.

/* Cara menulis warna */
.color-examples {
  /* Nama warna */
  color: red;
  
  /* Hexadecimal */
  background-color: #ff6b6b;
  
  /* RGB */
  border-color: rgb(255, 107, 107);
  
  /* RGBA (dengan transparency) */
  box-shadow: 0 0 10px rgba(255, 107, 107, 0.5);
  
  /* HSL */
  color: hsl(0, 79%, 72%);
}

/* Linear Gradient */
.gradient-linear {
  background: linear-gradient(45deg, #ff6b6b, #4ecdc4);
  background: linear-gradient(to right, red, blue, green);
}

/* Radial Gradient */
.gradient-radial {
  background: radial-gradient(circle, #ff6b6b, #4ecdc4);
}

12. CSS Animations & Transitions

CSS dapat membuat animasi dan transisi yang smooth tanpa JavaScript.

.transition-example {
  transition: all 0.3s ease;
  transition-property: transform, opacity;
  transition-duration: 0.5s;
  transition-timing-function: ease-in-out;
}

.transition-example:hover {
  transform: translateY(-10px) scale(1.1);
  opacity: 0.8;
}

/* Keyframe Animations */
@keyframes pulse {
  0% { transform: scale(1); opacity: 1; }
  50% { transform: scale(1.1); opacity: 0.7; }
  100% { transform: scale(1); opacity: 1; }
}

.animated-element {
  animation: pulse 2s infinite ease-in-out;
  animation-delay: 0.5s;
  animation-fill-mode: both;
}

13. Responsive Design

Responsive design memastikan website terlihat baik di semua ukuran layar menggunakan media queries.

/* Mobile First Approach */
.responsive-container {
  width: 100%;
  padding: 1rem;
}

/* Tablet */
@media (min-width: 768px) {
  .responsive-container {
    width: 80%;
    padding: 2rem;
  }
}

/* Desktop */
@media (min-width: 1024px) {
  .responsive-container {
    width: 1200px;
    margin: 0 auto;
    padding: 3rem;
  }
}

/* High Resolution Displays */
@media (min-resolution: 2dppx) {
  .high-res-image {
    background-image: url('image@2x.jpg');
  }
}
💡 Tip: Selalu gunakan meta viewport tag di HTML: <meta name="viewport" content="width=device-width, initial-scale=1.0">
